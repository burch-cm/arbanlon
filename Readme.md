## Arkosh’s Journal

I am trying to keep an in-game “journal”, written by the monk [Arkosh
Kovash](journal/session0.md). Arksoh doesn’t see or understand
everything that’s going on around him, so this isn’t an accurate account
of all events in the game. If you’d like to adjust entries due to
blatant factual errors, misspellings, etc., feel free to make edits and
submit a pull request. If you’d like to add information that Arkosh (or
I) don’t have, please consider starting your own journal for your
character and adding it here through a pull request.
